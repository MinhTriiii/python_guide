# How this guide works
- Start with `Basics.ipynb`. It provides a very detailed guide on how to start with Python (Jupyter notebooks) from the installation to some build-in functions of python 
- Git is not directly related to Python but I still provide some basic Git know-how that I use, as this is a beginner guide
- As the name suggests there is a roadmap of different Python packages that I want to explore (not necessarily in order). They are also listed here in this repository. In `Roadmap.ipynb` there is a small overview of the Python packages.
